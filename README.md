# WaferMap-Convert
Wafermap convert adalah proses menerjemahkan data peta wafer (lokasi chip baik/cacat pada piringan semikonduktor) dari satu format file atau standar ke format lain, seperti mengubah format biner lama ke format standar seperti SECS II atau GDSII untuk pembuatan mask atau integrasi data di jalur produksi pabrik.
