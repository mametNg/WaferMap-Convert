
# WaferMap Convert

Wafermap convert is the process of translating wafermap data (position of good/defective chips on a semiconductor wafer) from one file format or standard to another, e.g., converting older binary formats to standard formats such as SECS II or GDSII for mask generation or data integration in a factory production line.
